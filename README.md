# Retail Sales Analysis Using SQL

## 📌 Project Overview
This project analyzes retail sales data using SQL to identify business performance, profitable products, regional trends, and revenue optimization opportunities.

## 🎯 Objectives
- Analyze overall sales and profit performance
- Identify top-selling products
- Evaluate regional and category performance
- Detect loss-making products
- Generate business insights and recommendations

## 🛠 Tools Used
- MySQL
- SQL
- MySQL Workbench

## 📂 Dataset
The dataset contains:
- Order Date
- Product Name
- Category
- Region
- Quantity
- Sales
- Profit

## 📊 Key Analysis Performed
- Sales & Profit Analysis
- Product Performance Analysis
- Category Analysis
- Regional Performance
- Quantity Analysis
- Profit Margin Analysis
- Window Functions & Ranking
- Running Total Analysis

## 📈 Key Business Insights
- High-demand products contribute major revenue.
- Some high-sales products generate low profit margins.
- Regional performance varies significantly.
- Loss-making transactions were identified.

## ✅ Business Recommendations
- Focus marketing on high-profit products.
- Optimize pricing strategies.
- Improve performance in low-profit regions.
- Reduce operational losses.

## 🚀 SQL Skills Demonstrated
- GROUP BY
- Aggregations
- HAVING Clause
- Window Functions
- Ranking Functions
- Subqueries
- Business Analysis
